GiMU


GiMU is a game currently under development in Unity. Although it's similar in style to other RPG games, there are no limits to GiMU except for the incredibly long gaps in the development releases.

Controls


W: Forward
A: Left
S: Back
D: Right
Space: Jump
Shift: Sprint
Escape: Pause
These controls can be rebound in the menu when launching GiMU.

Where does one accuire GiMU?


One simply clicks Download at the top of this page. Or if you want source code, visit the github page.

What can you do?


For right now, the game is in a very early pre-alpha stage so basically you can walk around an island and fall through the world. In the final release, though, you'll be able to do basically anything. There's no real plot so the game is endless.

So basically it's Skyrim, right?


No. GiMU doesn't have arrows or knees to take said arrows to. (For now)
