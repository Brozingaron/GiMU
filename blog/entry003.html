System Requirements

November 13, 2012


Just a quick post. I've come up with a general list of system requirements.

To run GiMU at all, you'll need

  • A computer, monitor, keyboard and mouse
  • 100MB of free disk space
  • 256MB RAM
  • A recent CPU
  • A recent GPU with Direct X 9 Suppport
  • Windows XP or Mac OS X 10.6 (When the new build comes out)

If you want higher frame rates and graphics settings, you'll want

  • 2GB RAM
  • A dual-core CPU
  • A very nice integrated GPU or any dedicated GPU
  • A GPU that supports Direct X 11
  • Windows 7 or Mac OS X 10.8
  • A Mac made within the past 2 years, or one with a really nice GPU
For 64-bit Windows users, you can download the 64-bit version of the game if you're feeling daring, but the normal x86 version of the game will still run on 64-bit Windows.
Another suggestion I have is a bladder-releasing 1080p display, but it's totally awesome optional.

These completley scientific tests were performed in a very simple way. We tried the pre-alpha 1 build on a netbook, first-generation MacBook Air, budget laptop, slightly older epic laptop, and a slightly newer epic laptop. The netbook and Macbook Air were the only ones that lagged fullscreen. The others could play full screen with the highest graphics settings. Hooray simplicity!
