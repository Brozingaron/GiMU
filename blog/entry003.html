System Requirements

November 13, 2012


Just a quick post. I've come up with a general list of system requirements.

In order to run GiMU at all, you'll need

  • A computer, keyboard and mouse
  • 100MB of free disk space
  • 256MB RAM
  • A recent CPU
  • A recent GPU with Direct X 9 Suppport
  • Windows XP or Mac OS X 10.6 (When the new build comes out)

If you want higher frame rates and graphics settings, you'll want

  • 2GB RAM
  • A dual-core CPU
  • A very nice integrated GPU or any dedicated GPU
  • A GPU that supports Direct X 11
  • Windows 7 or Mac OS X 10.8
  • (So basically, any modern computer)
Another suggestion I have is a bladder-releasing 1080p display, but it's totally awesome optional.

These completley scientific tests were performed in a very simple way. We tried the pre-alpha 1 build on a netbook, budget laptop, slightly older epic laptop, and a slightly newer epic laptop. The netbook was the only one that lagged fullscreen at the highest graphic setting. Hooray simplicity!
